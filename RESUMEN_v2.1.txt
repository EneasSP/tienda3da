┌─ 💡 FÓRMULA DE COSTEO ────────────────────────────────────────────────────┐
│                                                                             │
│  COSTO FILAMENTO PLA:                                                      │
│    = (peso_gramos / 1000) × $45/kg                                         │
│                                                                             │
│  COSTO ENERGÍA (LUZ):                                                      │
│    = (tiempo_minutos / 60) × ($15/kWh × 3kW / 60)                         │
│                                                                             │
│  COSTO MÁQUINA (HORAS):                                                    │
│    = (tiempo_minutos / 60) × $150/hora                                     │
│                                                                             │
│  ─────────────────────────────────────────────────────────────             │
│  COSTO TOTAL = Filamento + Energía + Máquina                              │
│                                                                             │
│  GANANCIA (50%):                                                           │
│    = COSTO TOTAL × 0.5                                                     │
│                                                                             │
│  PRECIO VENTA = COSTO TOTAL + GANANCIA                                     │
│                                                                             │
│  📌 Ejemplo: Producto de 100g que tarda 60 minutos                         │
│     • Costo PLA: (100/1000) × 45 = $4.50                                   │
│     • Costo Luz: (60/60) × (15 × 3 / 60) = $0.75                           │
│     • Costo Máquina: (60/60) × 150 = $150.00                               │
│     • ─────────────────────────────────────                               │
│     • COSTO TOTAL = $155.25                                                │
│     • Ganancia 50% = $77.63                                                │
│     • PRECIO VENTA = $232.88                                               │
│                                                                             │
