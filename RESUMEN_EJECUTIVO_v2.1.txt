💡 FÓRMULA DE COSTEO (CORE DEL SISTEMA)

Entrada:     peso_gramos + tiempo_minutos

Cálculos:
  Costo Filamento  = (peso ÷ 1000) × $45/kg
  Costo Energía    = (tiempo ÷ 60) × $0.75/hora
  Costo Máquina    = (tiempo ÷ 60) × $150/hora
  ─────────────────────────────────────────
  COSTO TOTAL      = Suma anterior
  
  GANANCIA         = COSTO TOTAL × 50%
  
  PRECIO VENTA     = COSTO TOTAL + GANANCIA

Ejemplo:     100g + 60 minutos = $232.88

🚀 CÓMO USAR

1️⃣ APLICAR MIGRACIONES BD
   → Ejecutar: database/INSTRUCCIONES_MIGRACION_v2.1.sql
   → En phpMyAdmin o: mysql -u user -p db < archivo.sql

2️⃣ ACCEDER A CLIENTES
   → http://localhost:8000/clientes.php
   → Crear, buscar, editar, eliminar

3️⃣ ACCEDER A PRODUCTOS
   → http://localhost:8000/productos.php
   → PRUEBA EL CALCULADOR: cambia peso/tiempo y ve actualización

4️⃣ USAR API DE COSTO
   → GET /api/costo.php?peso=100&tiempo=60
   → Retorna JSON con desglose completo

5️⃣ EN MÓVIL
   → Hamburger menu automático
   → Interfaz adaptada
   → Botones touch-friendly
