🍔 HAMBURGER MENU - DETALLES

FUNCIONAMIENTO:
  1. Usuario hace clic en las 3 líneas
  2. Línea superior: rotate(45deg) + translateY(8px)
  3. Línea central: opacity: 0 (desaparece)
  4. Línea inferior: rotate(-45deg) + translateY(-8px)
  5. Resultado: Forma una "X"
  
  6. Menú desplegable aparece (max-height: 0 → 400px)
  7. Usuario hace clic en un tab
  8. Menú se cierra automáticamente
  9. Las líneas vuelven a forma de 3 líneas

ANIMACIÓN:
  • Duración: 0.3 segundos
  • Timing: ease (suave)
  • Sin saltos
  • Fluida y profesional

CSS PURO:
  • No requiere librerías JavaScript
  • Solo transform y opacity
  • GPU-accelerated (smooth)
  • Performance óptimo
